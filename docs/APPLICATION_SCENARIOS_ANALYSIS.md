---

## 🎯 场景分析

### 1. DApp 开发场景

**SDK 职责**：
- 提供客户端 API 封装
- 业务服务实现
- 交易构建和签名

**其他组件职责**：
- Contract SDK：智能合约开发
- Workbench：合约部署和测试
- 节点：交易执行和状态管理

---

### 2. 钱包应用场景

**SDK 职责**：
- 钱包功能实现
- 密钥管理
- 交易签名

**其他组件职责**：
- 节点：交易验证和执行
- 浏览器：用户界面

---

### 3. 后端服务场景

**SDK 职责**：
- 批量操作
- 高性能连接管理
- 业务服务封装

**其他组件职责**：
- 节点：交易执行
- 数据库：状态存储
